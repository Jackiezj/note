# HTTP请求

## HTTP携带参数的方式

| 参数位置 | 参数表现形式                                  | 参数提取方法              |
| -------- | --------------------------------------------- | ------------------------- |
| 请求行   | /weather/beijing?date=1                       | 正则提取(beijing, date=1) |
| 请求头   | Cookie, Content-Type                          |                           |
| 请求体   | 表单类型的字符串,xml格式字符串,json格式字符串 |                           |
